#### shared는 애플리케이션 전반에서 공통으로 사용되는 주요 UI 컴포넌트를 관리는 폴더

shared폴더 하위는 다음과 같은 구조를 가질수 있습니다!

#### shared 폴더 에서는 필요에 따라서 다음과 같은 하위 폴더를 만들수 있습니다.

### 하위 폴더

- **`api/`**
- **`constants/`**
- **`type/`**
- **`model/`**
- **`ui/`**

├── shared/ \
│ │ ├── api/ \
│ │ ├── constants/ \
│ │ ├── type/ \
│ │ ├── model/ \
│ │ └── ui/
